WEB易是一个chrome的扩展应用.

功能需求：
一，添加或修改项目
字段项：项目名称，项目域名(支持多个不同环境互相切换),项目描述信息。
显示要求：左侧选项卡显示项目列表与API列表，点击切换到API列表，右窗体域名等信息同时被改变。

二，添加或修改接口
字段项：API地址（不含域名），请求类型，GET参数，POST参数，HEADERS参数，每个参数说明，文本或文件、表达式值切换，返回值字段与结构，API描述，API编号并自动排序,接口开发进度（未启动，进行中，完成，异常），版本号。
功能开发：API请求发送，JSON结果格式化。
显示要求：右边窗体字段表单下显示添加和保存按钮，点击添加即可新增，否则修改当前。

三，错误码表添加与修改
字段项：错误代号，错误信息，代号文档说明。
显示要求：右窗体结果区上显示呼出错误码表，呼出的窗口可同时添加和修改错误码。
